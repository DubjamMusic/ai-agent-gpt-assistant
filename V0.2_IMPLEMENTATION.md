# GPT Agent v0.2 Implementation Guide

## Overview

This document describes the v0.2 implementation of the GPT Agent, which introduces a modular, extensible architecture with advanced capabilities for production use.

## Key Features Implemented

### 1. Modular Tool System (`tools.py`)

The tool system provides a plugin-based architecture for extending agent capabilities.

#### Features:
- **Tool Registry**: Centralized management of all available tools
- **Schema Generation**: Automatic conversion to OpenAI function calling format
- **Type Safety**: Parameter validation and type checking
- **Built-in Tools**: Pre-configured tools (time, calculator, web search)

#### Usage:

```python
from tools import ToolRegistry, ToolParameter

# Create a registry
registry = ToolRegistry()

# Define a custom tool
def my_tool(param1: str, param2: int) -> str:
    return f"Result: {param1} - {param2}"

# Register the tool
registry.register(
    name="my_tool",
    func=my_tool,
    description="A custom tool",
    parameters=[
        ToolParameter("param1", "string", "First parameter"),
        ToolParameter("param2", "integer", "Second parameter")
    ]
)

# Call the tool
result = registry.call_tool("my_tool", param1="test", param2=42)
```

### 2. Advanced Memory Management (`memory.py`)

Intelligent conversation memory with automatic summarization and persistence.

#### Features:
- **Conversation Turns**: Timestamped, metadata-rich message storage
- **Automatic Summarization**: Older conversations are summarized to manage token usage
- **Context Window Management**: Intelligently combines summaries and recent turns
- **Persistence**: Save/load conversation history to JSON
- **Statistics**: Track conversation metrics

#### Usage:

```python
from memory import MemoryManager

# Create memory manager
memory = MemoryManager(
    max_turns=50,
    summary_threshold=20,
    enable_persistence=True,
    persistence_file="conversation.json"
)

# Add turns
memory.add_turn("user", "Hello!")
memory.add_turn("assistant", "Hi there!")

# Get context for API calls
context = memory.get_context_window(max_tokens=4000)

# Get statistics
stats = memory.get_statistics()
print(f"Total turns: {stats['total_turns']}")
```

### 3. Refactored Agent with Async Support (`agent_v2.py`)

The core agent has been refactored to support multiple operation modes.

#### Features:
- **Synchronous Chat**: Traditional blocking chat interface
- **Asynchronous Chat**: Non-blocking async/await support
- **Streaming Responses**: Real-time response streaming
- **Function Calling**: Automatic tool invocation
- **Backward Compatibility**: v0.1 API wrapper for existing code

#### Usage:

```python
from agent_v2 import GPTAgentV2, AgentMode

# Create agent
agent = GPTAgentV2(
    model="gpt-3.5-turbo",
    system_prompt="You are a helpful assistant",
    enable_memory_persistence=True,
    mode=AgentMode.SYNC
)

# Synchronous chat
response = agent.chat("Hello!")

# Asynchronous chat
import asyncio
async def main():
    response = await agent.chat_async("Hello!")
    return response

# Streaming chat
for chunk in agent.stream_chat("Tell me a story"):
    print(chunk, end="", flush=True)
```

## Migration from v0.1 to v0.2

### Backward Compatibility

The v0.1 API is fully supported through a compatibility wrapper:

```python
# Old v0.1 code still works
from agent_v2 import GPTAgent

agent = GPTAgent(model="gpt-3.5-turbo")
agent.add_message("user", "Hello")
response = agent.chat("How are you?")
history = agent.get_history()
```

### Upgrading to v0.2

To use new v0.2 features:

```python
from agent_v2 import GPTAgentV2

# Use the new class
agent = GPTAgentV2(
    model="gpt-3.5-turbo",
    enable_memory_persistence=True,
    max_memory_turns=50
)

# Access new features
agent.register_tool("my_tool", my_func, "Description")
stats = agent.get_memory_stats()
```

## Advanced Usage

### Custom Tool Integration

```python
from agent_v2 import GPTAgentV2
from tools import ToolParameter

agent = GPTAgentV2()

# Define a complex tool
def database_query(sql: str) -> str:
    # Execute SQL and return results
    return "Query results..."

# Register it
agent.register_tool(
    name="query_db",
    func=database_query,
    description="Execute SQL queries on the database",
    parameters=[
        ToolParameter(
            name="sql",
            type="string",
            description="SQL query to execute",
            required=True
        )
    ]
)

# Agent can now use this tool
response = agent.chat("What are the top 10 users?")
```

### Conversation Persistence

```python
agent = GPTAgentV2(enable_memory_persistence=True)

# Have a conversation
agent.chat("Tell me about Python")
agent.chat("What about JavaScript?")

# Save conversation
agent.save_conversation("my_conversation.json")

# Later, load it back
new_agent = GPTAgentV2()
new_agent.load_conversation("my_conversation.json")

# Continue the conversation
response = new_agent.chat("What else should I learn?")
```

### Async Operations

```python
import asyncio
from agent_v2 import GPTAgentV2

async def main():
    agent = GPTAgentV2()
    
    # Run multiple chats concurrently
    responses = await asyncio.gather(
        agent.chat_async("What is AI?"),
        agent.chat_async("What is ML?"),
        agent.chat_async("What is DL?")
    )
    
    for response in responses:
        print(response)

asyncio.run(main())
```

## Performance Considerations

### Token Management

The memory manager automatically manages token usage by:
1. Keeping recent turns in active memory
2. Summarizing older conversations
3. Combining summaries with recent context

This allows for very long conversations without hitting token limits.

### Async Performance

For I/O-bound operations (API calls), async mode provides significant performance benefits:

```python
# Sequential (slow)
for msg in messages:
    response = agent.chat(msg)  # Waits for each response

# Concurrent (fast)
responses = await asyncio.gather(
    *[agent.chat_async(msg) for msg in messages]
)
```

## Troubleshooting

### "No OpenAI API key provided"

Set the `OPENAI_API_KEY` environment variable:

```bash
export OPENAI_API_KEY="your-key-here"
```

### Memory growing too large

Adjust the `max_memory_turns` and `summary_threshold` parameters:

```python
agent = GPTAgentV2(
    max_memory_turns=20,        # Smaller active memory
    summary_threshold=10        # Summarize more frequently
)
```

### Async operations not working

Ensure you're using `chat_async()` in an async context:

```python
import asyncio

async def main():
    agent = GPTAgentV2()
    response = await agent.chat_async("Hello")
    return response

asyncio.run(main())
```
